## ЧТО ЭТО ЗА ПРОЕКТ

Trends — первая Reels-платформа (вертикальное видео) внутри Telegram Mini App.
Лендинг — одностраничный инвестиционный сайт для привлечения Pre-Seed инвесторов.
Цель сбора: $1 000 000 на маркетинг и инфраструктуру платформы.

Живой MVP в Telegram: https://t.me/Trends_ibot?startapp

## СТЕК И АРХИТЕКТУРА

- Монорепо: pnpm workspaces
- Фронтенд: React + Vite + TypeScript (artifacts/trends-landing/)
- Стили: Tailwind CSS v4
- Анимации: framer-motion
- Роутинг: wouter (/ — лендинг, /cabinet — личный кабинет)
- Состояние: localStorage (бэкенда нет, всё на фронтенде)
- Иконки: lucide-react
- Компоненты UI: shadcn/ui (в src/components/ui/)

## ПРАВИЛА РАЗРАБОТКИ

- Язык интерфейса — только русский
- Без эмодзи в коде (только иконки lucide-react)
- Не импортировать React явно (JSX трансформер делает это сам)
- Хуки (useState, useEffect, useRef) — импортировать из "react"
- Все стейты — через localStorage, без API вызовов
- Анимации — framer-motion (motion.div + whileInView + variants)
- Для добавления новых иконок — проверить что они экспортируются из lucide-react
- Стили хранятся в index.css — НЕ менять CSS переменные :root

## КАК ЗАПУСКАТЬ ЛОКАЛЬНО (Replit)

Проект запускается автоматически через workflow "artifacts/trends-landing: web"
Он доступен в превью-панели Replit.

## КАК ДЕПЛОИТЬ НА RAILWAY

1. Запушь изменения в GitHub: Git → Commit → Push
2. Railway автоматически подхватит изменения (если настроен auto-deploy)
3. Или вручную: railway.app → твой проект → Deploy

Переменные на Railway:
  BASE_PATH = /

## ТИПИЧНЫЕ ПРАВКИ И КАК ИХ ДЕЛАТЬ

### Изменить текст в любом блоке
→ Открой artifacts/trends-landing/src/pages/Landing.tsx
→ Найди нужный текст (Ctrl+F) и поменяй

### Добавить нового члена команды
→ В Landing.tsx найди массив с командой (секция "BLOCK 7: КОМАНДА")
→ Добавь объект: { initials, name, role, roleColor, desc, tags, avatarColor, textColor }

### Изменить инвестиционный пакет
→ Открой artifacts/trends-landing/src/components/InvestmentModal.tsx
→ Найди массив PACKAGES и измени нужные поля

### Добавить новое изображение
→ Загрузи файл в attached_assets/ (перетащи в Replit)
→ Импортируй: import myImg from '@assets/имя_файла.png'
→ Используй: <img src={myImg} />

### Поменять ссылку на Telegram бот
→ В Landing.tsx найди href="https://t.me/Trends_ibot?startapp"

### Изменить цвет акцента
→ В src/index.css найди --primary и измени HSL значение
→ Текущее: 192 100% 50% (cyan #00D4FF)
