1. Сохрани токен как секрет GITHUB_TOKEN через requestEnvVar.

2. Клонируй репозиторий:
   git clone https://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@github.com/darcynj757-svg/trends-landing.git /tmp/repo

3. Скопируй файлы в workspace:
   cp -r /tmp/repo/artifacts/trends-landing artifacts/trends-landing
   mkdir -p attached_assets && cp /tmp/repo/attached_assets/* attached_assets/
   cp /tmp/repo/lib lib/ (если нужна lib)
   cp /tmp/repo/pnpm-workspace.yaml .
   cp /tmp/repo/tsconfig.base.json .
   cp /tmp/repo/package.json .
   cp /tmp/repo/pnpm-lock.yaml .

4. Создай артефакт через createArtifact:
   { artifactType: "react-vite", slug: "trends-landing", previewPath: "/", title: "Trends Landing" }

5. Установи зависимости:
   pnpm install

6. Настрой git для push обратно в GitHub:
   git remote set-url origin https://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@github.com/darcynj757-svg/trends-landing.git
   git config user.email "agent@replit.com"

7. Запусти воркфлоу "artifacts/trends-landing: web".
